## Como Executar

1. Clone este repositório
   ```sh
   git clone https://github.com/BillyTKD/Upload.git

2. Navegue até o diretório do projeto
    cd seu-repositorio.

3. Instale as dependências
    flutter pub get

4. Execute o aplicativo
    flutter run
